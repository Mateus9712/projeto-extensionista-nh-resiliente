# NH Resiliente - Sistema de Monitoramento e Educação Comunitária

Este projeto foi desenvolvido como parte da **Atividade Extensionista II** do curso de Análise e Desenvolvimento de Sistemas. O sistema tem como objetivo principal auxiliar na conscientização e no monitoramento de riscos climáticos e ambientais no município de Novo Hamburgo/RS.

## 🎯 Proposta do Projeto
A plataforma foi desenhada para aproximar a tecnologia da comunidade, oferecendo:
* **Monitoramento em Tempo Real (Simulado):** Integração com sistemas de verificação de nível do rio e alertas climáticos.
* **Mapeamento de Áreas de Risco:** Informações sobre áreas vulneráveis do município.
* **Educação e Preparo Comunitário:** Protocolos de emergência e instruções para montagem de kits de sobrevivência.

## 🛠️ Tecnologias Utilizadas
O desenvolvimento foi estruturado com foco em boas práticas e acessibilidade:
* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização moderna, responsiva e com modo escuro (*Dark Mode*) para facilitar a leitura noturna.
* **JavaScript:** Lógica de manipulação do DOM e simulação de dados e consumo de APIs.
* **Font Awesome:** Biblioteca de ícones para melhoria da interface visual.

## 🚀 Funcionalidades Técnicas Implementadas
1. **Dashboard de Monitoramento:** Exibe o nível do rio e a previsão do tempo atualizada.
2. **Sincronização de Dados:** Função assíncrona para simular a consulta de dados da Defesa Civil com feedback visual.
3. **Painel Dinâmico:** Exibição de informações sobre áreas de risco e kits de emergência sem recarregar a página.

## 🖥️ Como Executar o Projeto

Para visualizar o projeto em seu computador:
1. Faça o download ou clone este repositório.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estão na mesma pasta.
3. Dê um duplo clique no arquivo `index.html` ou abra-o em qualquer navegador moderno (Google Chrome, Microsoft Edge ou Mozilla Firefox).
